# SETUP GUIDE

## Prerequisites

Before running the application, ensure the following software is installed:

* Python 3.10 or higher
* Git
* Google Gemini API Key

Generate a free Gemini API key from:

https://aistudio.google.com/app/apikey

---

## Step 1: Clone the Repository

```bash
git clone <repository-url>
cd finreport-ai
```

---

## Step 2: Create a Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Step 3: Install Dependencies

Upgrade pip:

```bash
python -m pip install --upgrade pip
```

Install required packages:

```bash
pip install -r requirements.txt
```

Verify installation:

```bash
pip list
```

---

## Step 4: Configure Environment Variables

Create a file named:

```text
.env
```

Add your Gemini API key:

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

Example:

```env
GOOGLE_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

---

## Step 5: Run the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```

---

## Step 6: Using the Application

### Upload a Report

1. Open the application.
2. Upload a quarterly or annual report PDF.
3. Click **Process PDF**.
4. Wait until indexing is completed.

### Ask Questions

Examples:

```text
What was the total revenue this quarter?

Did the company declare a dividend?

What are the key risk factors?

What was the net profit after tax?
```

The application will:

1. Retrieve relevant report sections.
2. Generate a grounded answer using Gemini.
3. Display source citations with page numbers.

---

## Generated Files

After processing a report, the following files are created automatically:

```text
data/
├── index.faiss
├── chunks.pkl
└── metadata.json
```

Purpose:

| File          | Description            |
| ------------- | ---------------------- |
| index.faiss   | FAISS vector index     |
| chunks.pkl    | Stored document chunks |
| metadata.json | Document information   |

---

## Troubleshooting

### Missing API Key

Error:

```text
GOOGLE_API_KEY missing
```

Solution:

Verify that the `.env` file exists and contains a valid Gemini API key.

---

### No Indexed Document

Error:

```text
No indexed PDF found
```

Solution:

Upload and process a PDF before asking questions.

---

### Port Already in Use

Run Streamlit on another port:

```bash
streamlit run app.py --server.port 8502
```

---

### Module Not Found

Install dependencies again:

```bash
pip install -r requirements.txt
```
